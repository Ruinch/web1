# Mood & Habit Tracker

A student-friendly multi-page web project built with HTML, CSS, Bootstrap, and JavaScript.

## Pages
- `index.html` — home page with hero section and feature cards
- `form.html` — validated form with real-time feedback
- `dashboard.html` — localStorage-driven diary dashboard

## JavaScript files
- `js/form.js` — validation logic
- `js/ui.js` — mood theme, password toggle, animations
- `js/main.js` — submission, storage, and rendering
